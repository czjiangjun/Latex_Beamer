\frame{
	\frametitle{数据知识产权登记内容填写}
	\textcolor{magenta}{主要参照``数据知识产权登记平台操作指引''要求填写}
	\begin{itemize}
		\item \textcolor{red}{数据集名称}:~名称必须是``\underline{\textrm{XXXXX}\textcolor{purple}{数据集}}''
		\item \textcolor{blue}{数据来源}:~单位提供证明
		\item \textcolor{red}{数据格式}:~根据字段填写数据结构(有模板可参考)\footnote{填写的网页版数据结构用于公示时展示}
		\item \textcolor{red}{算法规则}:~需要提供算法说明文档(注明是否可以公开)
		\item \textcolor{red}{数据样例}:~网页页面填写不多于5条，数据样例文档中不多于100条
		\item \textcolor{blue}{数据知识产权登记承诺函}:~提交人撰写，加盖单位公章
	\end{itemize}
	审查时主要是格式审查，所以\textcolor{red}{红色标注的填写内容}，建议严格按照``操作指引''和``模板''的要求填写，减少反复修改、提交的周期
}
