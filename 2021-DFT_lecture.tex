%\setcounter{section}%{-1}% 控制chapter/section/page等的序号
现在开始介绍一下第一原理计算方法，考虑到根据现行的研究生教学课程设置，课上学习的主要是一些基本原理和理论，比如量子力学的基本原理，密度泛函理论。但是根据这些原理、理论形成的方法、算法和软件本身，涉及的不多。因为方法、算法实现会有比较高的数理要求，所以在此系统地梳理一下。在具体介绍各类方法之前，先了解一下有关计算物理和计算材料的背景知识。

\section{引言}
\subsection{第一页}
这是近年来比较流行的关于人类认知自然和科学活动“四种范式”的图示。
\begin{itemize}
	\item 第一阶段，大约一直延续到400年前，也就是\textrm{Galileo~Galilei}确立以实验观测研究自然现象和规律的近代科学研究思想之前，指导人们认知自然世界的，主要都是感性的日常经验以及先哲的箴言，所以主要都是经验科学，这个阶段的持续时间非常长
	\item 第二阶段，是近代社会自然科学的主要形式，就是将通过重复实验发现和总结的经验规律上升为基本原理，典型的比如\textrm{Newton}力学、热力学、以及\textrm{Maxwell}方程为代表的电动力学。这也是我们在教科书里学习的主要理性知识
	\item 第三阶段，是伴随着计算机和信息技术的出现而迅速发展起来的计算科学模式，也就是基于理论知识，通过计算机模拟，从微观和宏观尺度来预测、认知物质在特定条件下的运动规律，这是人类实验思想的一种延伸。计算机模拟对客观环境的依赖小
	\item 第四阶段，也是最新流行的数据驱动的科学研究模式，基于海量增长的研究数据，综合统计、数据挖掘和人工智能技术，深化对自然科学规律的认知层次。
\end{itemize}

\subsection{第二页}
回顾人类历史，不难看到社会的进步与材料的使用息息相关。随着使用和发现材料能力的提升，对各类材料的认识不断深化，人类已经距离追求从基本成分出发，通过对材料结果的设计，达到实现理想目标材料性能的梦想越来越近。

为了设计合理的材料性能，对物质不同尺度的物理规律必须有清晰的认识。为了研究材料在不同尺度的性能表现，科研工作者在不同的尺度范围开发了各种计算研究方法和工具(包括实验工具和理论模拟工具)，以实现对材料进行全方位的深入研究。

\subsection{第三页}
进入21世纪以后，欧洲、日本、美国等发达国家先后启动了以多尺度模拟、研发和科学设计为手段，提升材料设计成功率，缩短材料开发周期为目的的多种类型的国家级科研专项，其中以美国的“材料基因组计划\textrm{(Materials Genome Initiative， MGI)}”\cite{MGI_USA}最为著名。该计划的根本目的是通过增效集成各个尺度的材料模拟工具、高效实验手段和数据库，把材料研发从传统 经验式提升到科学设计，从而大大加快材料研发速度。我国也于2016年起启动了国家重点研发计划“材料基因工程关键技术与支撑平台”。

\subsection{第四页}
任何一个学科，其基础研究的软件，基本上都是这样一个架构:\\
由该学科的基本原理提炼的基本运动方程，在具体的应用环境下形成基本的理论，具体到第一原理计算，就是\textrm{Schr\"odinger}方程和密度泛函理论、能带理论；因为这类运动方程往往是偏微分方程，所以具体讨论如何求解此类方程时就需要一定的数理方程及其求解的背景知识，在我们的话题范围内就是基函数(具有原子特征)和势函数(周期势)的具体选择，以及空间积分的问题。在此基础上，借助经典的数值分析算法和计算机编程技术，最后形成一套可以用于实际的第一原理计算程序。

换句话说，如果你看过用过的计算软件，你将会发现，几乎所有的第一原理计算软件，都是照这个思想完成的，只要你相信基本方程、基本理论是对的，那么实际程序解出来的结果，是严格按照数值精度求解出来的结果，你就应该相信它。当然，如果你的模型和参数设置有问题，导致的结果不正确，另当别论。

\subsection{第五页}
这是一个完整的\textrm{DFT}方程求解过程的流程图，核心当然是\textrm{Kohn-Sham}方程的求解，这也将是我们后续讨论的核心，也就是不同计算方法下，具体的基函数如何构造、势函数如何选择，空间积分如何考虑，等等。而基函数的选择代表了不同方法的主要差异。周期势函数的选择也区分了普通分子计算和周期体系计算的主要区别。

\subsection{第六页}
这里简单看一下，对于周期体系计算，不同的方法(软件)对应的基函数和势函数的特点:
\begin{itemize}
	\item 赝势平面波
	\item 超软赝势
	\item \textrm{PAW}
	\item \textrm{LAPW}
\end{itemize}
